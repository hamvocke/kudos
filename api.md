Kudos REST API
==============

```
    GET     /feedback
    POST    /feedback
    GET     /feedback/<id>
    PUT     /feedback/<id>
    DELETE  /feedback/<id>

    GET     /feedback/<id>/votes
    POST    /feedback/<id>/votes
    GET     /feedback/<id>/votes/<id>
    PUT     /feedback/<id>/votes/<id>
    DELETE  /feedback/<id>/votes/<id>

    GET     /users
    POST    /users
    GET     /users/<id>
    PUT     /users/<id>
    DELETE  /users/<id>
```
